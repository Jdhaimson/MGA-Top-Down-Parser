SWI prolog port

  • docs: mgbeam.pdf
  • TD beam parser for SWI (tree drawing tools below!):
    representing a first example grammar: mg0.pl, lexBuild.pl,
    recognizer, parser: mgBeam.pl, mgBeamP.pl,
    transducers to see the trees the way you want them: beautify.pl, dt2st.pl, dt2stt.pl, dt2bt.pl, dt2xb.pl,
    more example grammars: mg-nt.pl, mg-nz.pl, mg-ne.pl, mgxx.pl
  • TD beam parser for old SWI: heaps.pl, mgbeamOldSWI.pl, mgbeampOldSWI.pl
  • To "pretty print" your trees, get this file: pp_tree.pl
    To use wishtree with SWI prolog, get these files: wish_tree.pl, draw_tree.pl, fonttbr12.pl
    (Instructions for getting tcltk to work on windows (thanks Victor!) are here)
